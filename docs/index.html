Get Started in Minutes

Three simple steps to improve your commit workflow

1

Install

npm install -g commitzero

Install CommitZero globally via npm or use it directly with npx

2

Configure

commitzero init

Initialize configuration and set up Git hooks for your project

3

Commit

git commit -m "feat: add new feature"

Start making conventional commits with automatic validation
