Valida tus Commits con CommitZero

Una poderosa herramienta CLI para hacer cumplir los estándares de Conventional Commits en tu flujo de trabajo Git. Sin dependencias de runtime, soporte completo de TypeScript.

Instalación Rápida

npm install -g commitzero

¿Por qué elegir CommitZero?

Construido para desarrolladores que se preocupan por la calidad del código y el historial consistente de Git

Conventional Commits

Hace cumplir la especificación de Conventional Commits con reglas de validación completas y configuración personalizable.

Sin Dependencias

Cero dependencias de runtime. Instalación ligera y rápida sin comprometer la funcionalidad.

CLI Amigable

Interfaz de línea de comandos intuitiva con mensajes de error claros y comandos interactivos.

Git Hooks

Integración automática con Git hooks para validación en tiempo real de commits y mensajes.

Configurable

Reglas completamente personalizables para adaptarse a las necesidades específicas de tu proyecto.

TypeScript

Soporte completo de TypeScript con tipos exportados para integración perfecta en tu proyecto.
