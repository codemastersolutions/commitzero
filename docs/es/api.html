Descripción General

CommitZero proporciona una interfaz programática completa para validar mensajes de commit, gestionar configuraciones e integrar Git hooks en sus aplicaciones y flujos de trabajo.

Características Principales

  • Validar mensajes de commit contra estándares de Conventional Commits
  • Analizar y formatear mensajes de commit
  • Cargar y gestionar archivos de configuración
  • Instalar y gestionar Git hooks
  • Soporte completo de TypeScript con definiciones de tipos

Instalación

Instale CommitZero como una dependencia en su proyecto para acceder a la API programática:

NPM 
npm install commitzero
Yarn 
yarn add commitzero
Importación Básica 
import { validate, parse, format } from 'commitzero';

// O usando CommonJS
const { validate, parse, format } = require('commitzero');

API Principal

La API principal proporciona funciones para validar, analizar y formatear mensajes de commit.

validate(message, options?)

Valida un mensaje de commit contra los estándares de Conventional Commits y reglas personalizadas.

Parámetros

Parámetro Tipo Descripción
message string El mensaje de commit a validar
options ValidationOptions? Configuración de validación opcional
Ejemplo de Uso 
import { validate } from 'commitzero';

const result = await validate('feat: agregar autenticación de usuario');

if (result.valid) {
  console.log('✅ Mensaje de commit válido');
} else {
  console.log('❌ Mensaje de commit inválido');
  result.errors.forEach(error => {
    console.log(`- ${error.message}`);
  });
}

Valor de Retorno

Devuelve un objeto ValidationResult:

interface ValidationResult {
  valid: boolean;
  errors: ValidationError[];
  warnings: ValidationWarning[];
  parsed?: ParsedCommit;
}

parse(message)

Analiza un mensaje de commit en sus partes constituyentes (tipo, alcance, descripción, cuerpo, pie de página).

Ejemplo de Uso 
import { parse } from 'commitzero';

const parsed = parse('feat(auth): agregar integración OAuth2

Implementar autenticación OAuth2 con proveedores Google y GitHub.

Closes #123');

console.log(parsed);
// {
//   type: 'feat',
//   scope: 'auth',
//   description: 'agregar integración OAuth2',
//   body: 'Implementar autenticación OAuth2 con proveedores Google y GitHub.',
//   footer: 'Closes #123'
// }

format(commit)

Formatea un objeto de commit analizado de vuelta a una cadena de mensaje de commit convencional.

Ejemplo de Uso 
import { format } from 'commitzero';

const commit = {
  type: 'feat',
  scope: 'auth',
  description: 'agregar integración OAuth2',
  body: 'Implementar autenticación OAuth2 con proveedores Google y GitHub.',
  footer: 'Closes #123'
};

const formatted = format(commit);
console.log(formatted);
// 'feat(auth): agregar integración OAuth2
//
// Implementar autenticación OAuth2 con proveedores Google y GitHub.
//
// Closes #123'

API de Configuración

Funciones para cargar y gestionar archivos de configuración de CommitZero.

loadConfig(path?)

Carga la configuración desde un archivo o busca configuración en ubicaciones estándar.

Ejemplo de Uso 
import { loadConfig } from 'commitzero';

// Cargar desde ubicaciones predeterminadas
const config = await loadConfig();

// Cargar desde ruta específica
const customConfig = await loadConfig('./custom-commitzero.config.js');

console.log(config.types); // Tipos de commit disponibles
console.log(config.scopes); // Alcances disponibles

createConfig(options)

Crea un nuevo archivo de configuración con las opciones especificadas.

Ejemplo de Uso 
import { createConfig } from 'commitzero';

await createConfig({
  path: './commitzero.config.js',
  types: ['feat', 'fix', 'docs', 'style', 'refactor'],
  scopes: ['api', 'ui', 'auth', 'db'],
  rules: {
    'subject-case': ['error', 'lower-case'],
    'subject-max-length': ['error', 50]
  }
});

API de Hooks

Funciones para gestionar la integración de Git hooks.

installHooks(options?)

Instala Git hooks para validación automática de mensajes de commit.

Ejemplo de Uso 
import { installHooks } from 'commitzero';

// Instalar con opciones predeterminadas
await installHooks();

// Instalar con opciones personalizadas
await installHooks({
  hooks: ['commit-msg', 'pre-commit'],
  gitDir: './.git',
  force: true
});

uninstallHooks(options?)

Elimina los Git hooks de CommitZero del repositorio.

Ejemplo de Uso 
import { uninstallHooks } from 'commitzero';

// Desinstalar todos los hooks
await uninstallHooks();

// Desinstalar hooks específicos
await uninstallHooks({
  hooks: ['commit-msg'],
  gitDir: './.git'
});

Tipos de TypeScript

CommitZero incluye definiciones de tipos de TypeScript completas para todas las funciones de API y estructuras de datos.

Tipos Principales 
interface ParsedCommit {
  type: string;
  scope?: string;
  description: string;
  body?: string;
  footer?: string;
  breaking?: boolean;
}

interface ValidationResult {
  valid: boolean;
  errors: ValidationError[];
  warnings: ValidationWarning[];
  parsed?: ParsedCommit;
}

interface ValidationError {
  message: string;
  rule: string;
  level: 'error' | 'warning';
}
Tipos de Configuración 
interface CommitZeroConfig {
  types: string[];
  scopes?: string[];
  rules: Record;
  parserPreset?: string;
  formatter?: string;
  helpUrl?: string;
}

interface ValidationOptions {
  config?: CommitZeroConfig;
  cwd?: string;
  from?: string;
  to?: string;
}

Ejemplos Completos

Herramienta de Validación Personalizada

validate-commits.js 
#!/usr/bin/env node

import { validate, loadConfig } from 'commitzero';
import { readFileSync } from 'fs';
import { execSync } from 'child_process';

async function validateCommits() {
  try {
    // Cargar configuración
    const config = await loadConfig();
    
    // Obtener mensajes de commit del git log
    const commits = execSync('git log --oneline -10 --pretty=format:%s')
      .toString()
      .split('\n')
      .filter(Boolean);

    let hasErrors = false;

    for (const commit of commits) {
      const result = await validate(commit, { config });
      
      if (!result.valid) {
        console.log(`❌ ${commit}`);
        result.errors.forEach(error => {
          console.log(`   ${error.message}`);
        });
        hasErrors = true;
      } else {
        console.log(`✅ ${commit}`);
      }
    }

    if (hasErrors) {
      process.exit(1);
    }
  } catch (error) {
    console.error('Error validando commits:', error.message);
    process.exit(1);
  }
}

validateCommits();

Integración CI/CD

Flujo de Trabajo de GitHub Actions 
name: Validar Commits

on:
  pull_request:
    branches: [main]

jobs:
  validate-commits:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
        with:
          fetch-depth: 0
      
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      
      - run: npm install commitzero
      
      - name: Validar mensajes de commit
        run: |
          node -e "
          import { validate } from 'commitzero';
          import { execSync } from 'child_process';
          
          const commits = execSync('git log origin/main..HEAD --pretty=format:%s')
            .toString().split('\n').filter(Boolean);
          
          let hasErrors = false;
          for (const commit of commits) {
            const result = await validate(commit);
            if (!result.valid) {
              console.log('❌', commit);
              result.errors.forEach(e => console.log('  ', e.message));
              hasErrors = true;
            }
          }
          if (hasErrors) process.exit(1);
          "
