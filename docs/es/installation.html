Instalación

Aprende cómo instalar CommitZero en tu entorno de desarrollo. Hay múltiples métodos de instalación disponibles para adaptarse a tu flujo de trabajo.

Requisitos del Sistema

Requisito de Node.js

CommitZero requiere Node.js versión 16 o superior. Asegúrate de tener una versión compatible instalada antes de continuar.

Verificar Versión de Node.js

node --version

Verificar Versión de NPM

npm --version

Instalación con NPM

La forma más común de instalar CommitZero es a través de NPM, el gestor de paquetes de Node.js.

Instalación Global

Instala CommitZero globalmente para usarlo en cualquier proyecto.

npm install -g commitzero

Recomendado para uso general y múltiples proyectos

Instalación Local

Instala CommitZero como dependencia de desarrollo en tu proyecto.

npm install --save-dev commitzero

Ideal para proyectos específicos y control de versiones

Instalación con Yarn

Si prefieres usar Yarn como gestor de paquetes, también puedes instalar CommitZero con él.

Instalación Global con Yarn

yarn global add commitzero

Instalación Local con Yarn

yarn add --dev commitzero

Instalación con PNPM

PNPM es un gestor de paquetes rápido y eficiente que también es compatible con CommitZero.

Instalación Global con PNPM

pnpm add -g commitzero

Instalación Local con PNPM

pnpm add -D commitzero

Uso con NPX

Puedes usar CommitZero sin instalarlo permanentemente usando NPX, que viene incluido con NPM.

Ejecutar sin Instalación

Ejecuta CommitZero directamente sin instalarlo en tu sistema:

npx commitzero --help

Nota: NPX descargará y ejecutará la última versión cada vez, lo que puede ser más lento que una instalación local.

Verificación de la Instalación

Después de la instalación, verifica que CommitZero esté correctamente instalado y funcionando.

Verificar Versión

Comprueba que CommitZero esté instalado y muestra su versión:

commitzero --version

Mostrar Ayuda

Muestra la información de ayuda y comandos disponibles:

commitzero --help

¡Instalación Exitosa!

Si ambos comandos funcionan correctamente, CommitZero está instalado y listo para usar. Continúa con la configuración para personalizar tu experiencia.

Solución de Problemas

Soluciones a problemas comunes que puedes encontrar durante la instalación.

Comando no encontrado

Si obtienes un error "command not found" después de la instalación global:

bash: commitzero: command not found

Solución 1: Verificar PATH

Asegúrate de que el directorio global de NPM esté en tu PATH:

npm config get prefix

Solución 2: Reinstalar

Intenta desinstalar y reinstalar:

npm uninstall -g commitzero && npm install -g commitzero

Errores de Permisos

Si encuentras errores de permisos durante la instalación global:

Solución 1: Usar sudo (macOS/Linux)

sudo npm install -g commitzero

Solución 2: Configurar directorio NPM

Configura NPM para usar un directorio diferente:

npm config set prefix ~/.npm-global

Versión de Node.js Incompatible

Si tu versión de Node.js es menor a 16:

Actualizar Node.js

Visita nodejs.org para descargar la última versión LTS.

Usar NVM (recomendado)

Instala y usa Node Version Manager para gestionar múltiples versiones:

nvm install --lts && nvm use --lts

Desinstalación

Si necesitas desinstalar CommitZero, aquí tienes los comandos para cada método de instalación.

Desinstalación Global

NPM

npm uninstall -g commitzero

Yarn

yarn global remove commitzero

PNPM

pnpm remove -g commitzero

Desinstalación Local

NPM

npm uninstall commitzero

Yarn

yarn remove commitzero

PNPM

pnpm remove commitzero

Limpiar Configuración

Después de desinstalar, puedes eliminar manualmente cualquier archivo de configuración de CommitZero (como .commitzerorc.json) de tus proyectos si ya no los necesitas.
