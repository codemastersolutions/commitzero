v2.0.0

Cambios Importantes 15 de noviembre, 2024

Cambios Importantes

Requisitos de Node.js actualizados

Ahora requiere Node.js 18 o superior.

# Antes: Node.js 14+
# Ahora: Node.js 18+

Formato de configuración actualizado

Los archivos de configuración ahora deben usar formato JavaScript en lugar de JSON.

// Antes: commitzero.json
// Ahora: commitzero.config.js

Bandera --legacy eliminada

La bandera --legacy ha sido eliminada. Usa el nuevo formato de configuración.

API validate() ahora es asíncrona

La función validate() ahora devuelve una Promise.

// Antes
const result = validate(message);

// Ahora
const result = await validate(message);

Nuevas Características

Tipos de commit personalizados

Define tus propios tipos de commit con descripciones personalizadas.

module.exports = {
  types: {
    custom: { description: 'Cambio personalizado' }
  }
};

Scopes personalizados

Define scopes específicos para tu proyecto con validación automática.

Modo interactivo mejorado

Nueva interfaz interactiva con autocompletado y sugerencias inteligentes.

Git hooks mejorados

Instalación automática y gestión mejorada de Git hooks.

Definiciones TypeScript

Soporte completo de TypeScript con definiciones de tipos incluidas.

Bandera --config

Especifica archivos de configuración personalizados con --config.

Mejoras

  • Mejoras significativas de rendimiento (hasta 3x más rápido)
  • Mensajes de error más claros y útiles
  • Salida CLI mejorada con colores y formato
  • Documentación mejorada con más ejemplos
