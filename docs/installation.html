Installation

Learn how to install CommitZero in your development environment. Multiple installation methods are available to suit your workflow.

System Requirements

Node.js Requirement

CommitZero requires Node.js version 16 or higher. Make sure you have a compatible version installed before proceeding.

Check Node.js Version

node --version

Check NPM Version

npm --version

NPM Installation

The most common way to install CommitZero is through NPM, the Node.js package manager.

Global Installation (Recommended)

Install CommitZero globally to use it across all your projects:

npm install -g commitzero

Recommended: Global installation allows you to use commitzero command from anywhere in your system.

Local Installation

Install CommitZero as a development dependency in your project:

npm install --save-dev commitzero

When installed locally, use npx commitzero or add scripts to your package.json.

Yarn Installation

If you prefer Yarn as your package manager, you can install CommitZero using these commands:

Global Installation

yarn global add commitzero

Local Installation

yarn add -D commitzero

PNPM Installation

PNPM is a fast, disk space efficient package manager. Install CommitZero with these commands:

Global Installation

pnpm add -g commitzero

Local Installation

pnpm add -D commitzero

NPX Usage (No Installation)

You can use CommitZero without installing it globally by using NPX:

One-time Usage

npx commitzero --help

Note: NPX will download and run the latest version each time. For better performance, consider global installation.

Verify Installation

After installation, verify that CommitZero is working correctly:

Check Version

commitzero --version

This should display the installed version of CommitZero.

View Help

commitzero --help

This will show all available commands and options.

Troubleshooting

Common installation issues and their solutions:

Permission Errors (macOS/Linux)

If you encounter permission errors during global installation, try using sudo:

sudo npm install -g commitzero

Command Not Found

If the commitzero command is not found after global installation:

  • Check if npm global bin directory is in your PATH
  • Run npm config get prefix to see the global directory
  • Add the bin directory to your PATH environment variable
  • Restart your terminal

Node.js Version Issues

If you're using an older version of Node.js:

  • Update Node.js to version 16 or higher
  • Use a Node.js version manager like nvm
  • Check the official Node.js website for installation instructions

Uninstallation

If you need to remove CommitZero from your system:

Global Uninstall

npm uninstall -g commitzero

For NPM installations

Local Uninstall

npm uninstall commitzero

From project dependencies
