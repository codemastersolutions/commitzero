Uso Básico

✅ Mensagens de Commit Válidas

# Nova funcionalidade
feat: adicionar autenticação OAuth

# Correção de bug
fix: corrigir vazamento de memória no cache

# Documentação
docs: atualizar guia de instalação

# Com escopo
feat(auth): implementar login com Google
fix(api): corrigir validação de entrada
docs(readme): adicionar exemplos de uso

# Com breaking change
feat!: migrar para nova API de autenticação
BREAKING CHANGE: remove suporte para tokens v1

# Com issue referenciada
fix: corrigir erro de validação

Closes #123

❌ Mensagens de Commit Inválidas

# Sem tipo
adicionar nova funcionalidade

# Tipo inválido
update: melhorar performance

# Sem dois pontos
feat adicionar login

# Assunto muito longo
feat: adicionar uma nova funcionalidade super complexa que faz muitas coisas diferentes e tem uma descrição muito longa

# Assunto vazio
feat: 

# Primeira letra maiúscula
feat: Adicionar nova funcionalidade

# Ponto final no assunto
feat: adicionar nova funcionalidade.

Integração com CI/CD

GitHub Actions

Exemplo de workflow para validar commits em Pull Requests: 

# .github/workflows/validate-commits.yml
name: Validar Commits

on:
  pull_request:
    branches: [ main, develop ]

jobs:
  validate-commits:
    runs-on: ubuntu-latest
    
    steps:
    - name: Checkout
      uses: actions/checkout@v4
      with:
        fetch-depth: 0
    
    - name: Setup Node.js
      uses: actions/setup-node@v4
      with:
        node-version: '18'
        cache: 'npm'
    
    - name: Instalar CommitZero
      run: npm install -g commitzero
    
    - name: Validar commits do PR
      run: |
        # Obter lista de commits do PR
        COMMITS=$(git rev-list --reverse origin/${{ github.base_ref }}..${{ github.head_ref }})
        
        # Validar cada commit
        for commit in $COMMITS; do
          echo "Validando commit: $commit"
          MESSAGE=$(git log --format=%B -n 1 $commit)
          echo "$MESSAGE" | commitzero validate --stdin
        done
    
    - name: Comentar no PR se houver erros
      if: failure()
      uses: actions/github-script@v7
      with:
        script: |
          github.rest.issues.createComment({
            issue_number: context.issue.number,
            owner: context.repo.owner,
            repo: context.repo.repo,
            body: '❌ **Validação de Commits Falhou**\n\nAlguns commits não seguem as convenções. Verifique os logs da action para mais detalhes.'
          })

GitLab CI

# .gitlab-ci.yml
stages:
  - validate

validate-commits:
  stage: validate
  image: node:18-alpine
  only:
    - merge_requests
  before_script:
    - npm install -g commitzero
  script:
    - |
      # Obter commits do MR
      COMMITS=$(git rev-list --reverse $CI_MERGE_REQUEST_TARGET_BRANCH_SHA..$CI_COMMIT_SHA)
      
      # Validar cada commit
      for commit in $COMMITS; do
        echo "Validando commit: $commit"
        MESSAGE=$(git log --format=%B -n 1 $commit)
        echo "$MESSAGE" | commitzero validate --stdin
      done
  allow_failure: false

Script de Setup para Novos Desenvolvedores

#!/bin/bash
# setup-commitzero.sh

echo "🚀 Configurando CommitZero para o projeto..."

# Instalar CommitZero globalmente
echo "📦 Instalando CommitZero..."
npm install -g commitzero

# Verificar se já existe configuração
if [ ! -f ".commitzero.json" ]; then
  echo "⚙️ Criando configuração padrão..."
  commitzero init --preset=team
fi

# Instalar hooks Git
echo "🪝 Instalando hooks Git..."
commitzero install

# Configurar Git template (opcional)
echo "📝 Configurando template de commit..."
cat > .gitmessage << 'EOF'
# <tipo>[escopo opcional]: <descrição>
#
# [corpo opcional]
#
# [rodapé opcional]
#
# Tipos disponíveis:
# feat: nova funcionalidade
# fix: correção de bug
# docs: documentação
# style: formatação
# refactor: refatoração
# test: testes
# chore: tarefas gerais
# perf: performance
# ci: CI/CD
# build: sistema de build
# revert: reverter commit
EOF

git config commit.template .gitmessage

echo "✅ CommitZero configurado com sucesso!"
echo ""
echo "💡 Dicas:"
echo "  - Use 'commitzero' para commits interativos"
echo "  - Use 'git commit' normalmente (validação automática)"
echo "  - Veja exemplos em: https://commitzero.github.io/commitzero/pt-br/examples"
