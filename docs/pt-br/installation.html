Instalação

Instale CommitZero em seu sistema e comece a validar commits em minutos

Requisitos do Sistema

Antes de começar

  • • Node.js 14.0 ou superior
  • • npm, yarn ou pnpm
  • • Git instalado em seu sistema

Instalação via NPM

Instalação Global (Recomendada)

Instale CommitZero globalmente para usar em qualquer projeto: 

npm install -g commitzero

Instalação Local

Instale CommitZero como dependência de desenvolvimento em um projeto específico: 

npm install --save-dev commitzero

Instalação via Yarn

Instalação Global

yarn global add commitzero

Instalação Local

yarn add --dev commitzero

Instalação via PNPM

Instalação Global

pnpm add -g commitzero

Instalação Local

pnpm add -D commitzero

Uso via NPX (Sem Instalação)

Use CommitZero sem instalar permanentemente: 

npx commitzero --help

Verificar Instalação

Após a instalação, verifique se CommitZero foi instalado corretamente: 

commitzero --version

Você também pode ver todas as opções disponíveis: 

commitzero --help

Solução de Problemas

Erro de Permissão (EACCES)

Se você receber um erro de permissão ao instalar globalmente, tente uma das seguintes soluções:

Opção 1: Use sudo (Linux/macOS)

sudo npm install -g commitzero

Opção 2: Configure um diretório npm diferente

mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH

Comando Não Encontrado

Se o comando commitzero não for encontrado após a instalação:

  • • Verifique se o diretório global do npm está no seu PATH
  • • Reinicie seu terminal
  • • Use npx commitzero como alternativa

Versão do Node.js

CommitZero requer Node.js 14.0 ou superior. Verifique sua versão: 

node --version

Desinstalação

Para remover CommitZero do seu sistema:

NPM Global

npm uninstall -g commitzero

Yarn Global

yarn global remove commitzero

PNPM Global

pnpm remove -g commitzero
