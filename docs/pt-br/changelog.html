Changelog

Acompanhe todas as mudanças, melhorias e correções do CommitZero

v2.0.0

15 de Janeiro, 2024

Mudanças Incompatíveis

  • Removido suporte para Node.js < 16.0.0
  • Alterado formato do arquivo de configuração (migração automática disponível)
  • API validate() agora retorna Promise em vez de resultado síncrono
  • Renomeado comando --setup para --init

Novas Funcionalidades

  • Suporte completo para TypeScript com tipos exportados
  • Modo interativo melhorado com autocomplete e sugestões
  • Integração nativa com editores (VS Code, Vim, Emacs)
  • Suporte para configuração em YAML e TOML
  • Plugin system para extensões personalizadas
  • Validação em tempo real durante digitação

Melhorias

  • Performance 3x mais rápida na validação de commits
  • Interface de linha de comando redesenhada
  • Mensagens de erro mais claras e acionáveis
  • Documentação completamente reescrita

v1.5.0

28 de Novembro, 2023

Novas Funcionalidades

  • Adicionado suporte para monorepos com configurações por workspace
  • Comando commitzero format para reformatar mensagens existentes
  • Integração com GitHub Actions e GitLab CI
  • Suporte para templates de commit personalizados

Melhorias

  • Melhor detecção de repositórios Git
  • Suporte aprimorado para Windows
  • Validação mais robusta de escopos

v1.4.2

15 de Outubro, 2023

Correções de Bugs

  • Corrigido crash ao usar escopos com caracteres especiais
  • Resolvido problema de encoding em sistemas Windows
  • Corrigida validação incorreta de breaking changes
  • Melhorada compatibilidade com Git 2.40+

v1.4.0

2 de Setembro, 2023

Novas Funcionalidades

  • Comando commitzero interactive para criação guiada de commits
  • Suporte para configuração global do usuário
  • Validação de comprimento de linha configurável
  • Modo silencioso com flag --quiet

Melhorias

  • Interface mais intuitiva e colorida
  • Melhor tratamento de erros
  • Documentação expandida com mais exemplos

v1.3.1

18 de Agosto, 2023

Correções de Bugs

  • Corrigido problema com hooks em repositórios bare
  • Resolvida falha na detecção de configuração em subdiretórios
  • Melhorada compatibilidade com npm workspaces

v1.3.0

5 de Julho, 2023

Novas Funcionalidades

  • Suporte para Git hooks automáticos (pre-commit, commit-msg)

  • Suporte para múltiplos tipos de commit por projeto

Melhorias

  • Suporte aprimorado para diferentes shells
