Track all changes, new features, bug fixes, and improvements across CommitZero versions.

v2.0.0

Breaking Changes December 15, 2024

Breaking Changes

  • Minimum Node.js version is now 18.0.0
  • Configuration file format changed from JSON to JavaScript
  • Removed deprecated --legacy flag
  • API: validate() now returns Promise

New Features

  • Added support for custom commit types and scopes
  • New interactive mode with guided commit creation
  • Enhanced Git hooks with pre-commit validation
  • Added TypeScript definitions for better IDE support
  • New --config flag to specify custom config file

Improvements

  • Performance improvements: 40% faster validation
  • Better error messages with suggestions
  • Improved CLI output with colors and formatting
  • Enhanced documentation and examples

New Features

Bug Fixes

  • Resolved configuration loading in nested directories
  • Fixed Git hooks installation on macOS
